9. Scheduled Section — Column Labels in Separator

The dates sit in their own visual column (between title and duration). The problem: no header tells you what that column is. Solution: embed column labels INTO the section separator line.

Proposed — labels embedded in the line

The separator's trailing line becomes a grid that mirrors the task columns. Column labels sit ON the line with a background knockout.

How it works: The separator's trailing line becomes a grid matching the task columns below. Labels ("Date", "Dur", "Impact", "Clarity") sit ON the line with a white background knockout — they look like they're part of the separator, not a separate header row. The leading portion of the line (above the title area) stays empty. Column alignment is automatic because both separator and task rows share the same grid column widths.

Date color: Red = overdue · Purple = today · Muted = future
