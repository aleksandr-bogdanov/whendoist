Redesign Mockup

Apple Glass + Telegram feel for Whendoist. Toggle each change to see its effect on the phone preview. All fonts support Cyrillic.

1. Typography

Quicksand was chosen for rounded terminals matching the W icon. But consider: Rubik has soft rounded corners + excellent Cyrillic, and Nunito is Quicksand's spiritual sibling with tighter metrics. Inter is the Apple/system font aesthetic ‚Äî clean, geometric, precise.

2. Spacing

Current: 7px vertical padding on task rows. Telegram uses ~14px. Brand says "clear visual breathing room" ‚Äî more padding is compatible.

3. Impact Row Wash

Currently each task has a faint colored background tint per impact level. Telegram uses plain white. Removing wash gives a cleaner look. Impact info is already in the text column + rail.

4. Task Separators

Telegram uses thin hairlines between list items. Currently tasks have no separators ‚Äî the row wash provides visual grouping. With wash removed, separators become essential.

5. Impact Indicator

The 4px left rail is a strong visual element. Options: keep as-is, thin it to 2px at lower opacity, or replace with a small dot before the task title (Telegram notification-badge style). Brand says "tint-based interactions" ‚Äî a dot is more subtle.

6. Sort Header

Current: ALL CAPS with letter-spacing. Apple/Telegram use sentence case and lighter weight. Less "spreadsheet", more "app".

7. Add Task Row

Dashed border is dated. Clean version: just subtle text, no border. FAB already exists for adding tasks.

8. Domain Headers

Slightly bolder + larger domain names create clearer visual hierarchy. Compatible with brand principle "separation through boldness".
