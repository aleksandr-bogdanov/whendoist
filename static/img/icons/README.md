# Whendoist Icon System

> SVG icons for the Whendoist UI. All icons use a 24Ã—24 viewBox with 2px stroke width.

## Usage

```html
<!-- Inline SVG (recommended) -->
<svg class="icon" viewBox="0 0 24 24" width="20" height="20">
  <use href="/static/img/icons/ui-icons.svg#edit"/>
</svg>

<!-- Or copy the path directly -->
<svg viewBox="0 0 24 24" width="16" height="16" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">
  <!-- paths here -->
</svg>
```

## Standard Sizes

| Size | Use Case |
|------|----------|
| 12px | Inline with small text |
| 16px | Buttons, chips |
| 20px | Standard UI elements |
| 24px | Headers, emphasis |

## Categories

- **Actions**: edit, delete, close, plus, check, copy, download, upload
- **Navigation**: menu, chevron-down/up/left/right, arrow-left/right, external-link
- **Objects**: calendar, clock, task, folder, lock, unlock, key, shield
- **Status**: spinner, check-circle, x-circle, alert, info
- **Features**: energy, thought, analytics, settings, search, filter, sort

See `docs/brand/icon-reference.html` for the complete interactive reference.
